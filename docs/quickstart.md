# Quickstart 🌱

A 60-second tour of SolCoach.

```
   ┌──────────────┐      ┌──────────────┐      ┌──────────────┐
   │  Onboarding  │  ──▶ │ Daily Quest  │  ──▶ │   Streaks    │
   │  pick a role │      │ accept/skip  │      │  +1 each day │
   └──────────────┘      └──────────────┘      └──────────────┘
        ↓                      ↓                      ↓
    8 roles to try       fresh quest 24h       streak shield 🛡
```

## Get going

1. Connect Phantom (devnet) — your wallet picks your hero icon
2. Pick a role on the onboarding screen (Trader, Builder, Yield Farmer…)
3. Tap *Accept* on today's quest
4. Come back tomorrow — the streak counter ticks up 🔥

## What lights up the streak

- Accept the daily task before the countdown ring closes
- Complete it (we check on-chain via your wallet)
- Miss a day? Streak shield 🛡 saves you once a week

## Where to find things

| Page         | What it shows                          |
|--------------|-----------------------------------------|
| `/`          | Today's quest, streak, level path      |
| `/heroes`    | Top streaks across all coaches         |
| `/history`   | Your past quests & completion record   |
| `/profile`   | Wallet, role, achievements             |
